- Se connecter A la VM linux avec le super user root et cré le compte "admin"
  # adduser admin

- Se connecter avec le user "admin" ( avec chargement du .profil / $HOME ) --> "su - admin" ( au lieu de "su admin" )
  # su - admin 

- Cré le rértoire " SCRIPTS " dans /home/admin
  # mkdir SCRIPTS

- Cré les rértoires suivants dans "/home/admin" : app, sauvegardes, my_projects, archives.
  # mkdir app sauvegardes my_projects archives

- Cré un script nommécreate_directories.sh" dans le rértoire SCRIPTS et accordez les droits d'exétion seulement pour le compte admin.
  # cd SCRIPTS
  # vi create_directories.sh
  # chmod u+x create_directories.sh
